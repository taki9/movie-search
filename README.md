# movie-search

Search component based on the autocomplete of MUI

## [Live storybook](https://cranky-hermann-d1f0dc.netlify.app)

## Start on local env

Intall dependencies:

```
yarn
```

Run the storybook in dev mode:

```
yarn dev
```

## Unit tests

To run the unit tests use the following command:

```
yarn test
```
